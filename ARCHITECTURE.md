## 命令列表

| 命令 | 功能 |
|------|------|
| `/start` | 显示帮助信息 |
| `/discuss <话题>` | 手动启动圆桌讨论 |
| `/stop` | 立即终止讨论和所有AI进程 |
| `/ls` | 列出项目文件 |
| `/clear` | 清空所有会话和对话历史 |
| `/clear_session [AI]` | 清空指定AI的会话 |
| `/sessions` | 查看当前会话状态 |

## 特殊语法

### 指定历史条数
```
--N <消息内容>
```
- 默认传递最近 **2条** 对话历史
- 使用 `--5 codex帮我看看` 传递最近5条
- 范围限制: 1-20条

### AI调用方式
```
@claude 问题           # 明确调用
claude: 问题           # 冒号调用
codex和gemini 问题     # 并行调用
讨论一下...            # 触发圆桌讨论
```

### 引用 vs 调用
```
codex你觉得claude说的对吗?
       ↓
只触发 Codex, 不触发 Claude (因为"claude说的"是引用)
```

## 错误处理机制

### 统一错误消息

所有 AI 执行错误使用标准化的错误常量（`agents/runner.py`）：

```python
ERROR_SESSION_EXPIRED    # 会话过期或无效
ERROR_NO_RESPONSE        # AI无响应（可能网络问题）
ERROR_TIMEOUT           # 响应超时
ERROR_CANCELLED         # 操作被用户中断
ERROR_GENERAL           # 通用错误
```

### 错误处理流程

```
AI CLI 执行
    │
    ├─ returncode != 0  ──► 检查错误类型
    │                       │
    │                       ├─ Session相关 ──► 清除会话 + 返回 SESSION_EXPIRED
    │                       └─ 其他错误 ──────► 返回 GENERAL
    │
    ├─ 空响应 ──────────────► 返回 NO_RESPONSE
    │
    ├─ 超时 ────────────────► 返回 TIMEOUT
    │
    └─ 取消 ────────────────► 返回 CANCELLED
```

## 进程管理机制

### 进程追踪

- **活跃进程字典**: `active_processes = {(chat_id, agent_name): Process}`
- **并发保护**: 使用 `asyncio.Lock` 保护字典访问
- **注册时机**: 进程创建后立即注册
- **清理时机**:
  - 进程正常结束后自动清理
  - 用户 `/stop` 命令
  - 程序退出时（信号处理器）

### 优雅退出流程

```
Ctrl+C 或 SIGTERM
    │
    ▼
signal_handler
    │
    ├─ 停止 Telegram Bot
    │
    ├─ 保存会话数据（立即，不防抖）
    │
    ├─ 终止所有活跃的 AI 进程
    │  （精确追踪，不使用 pkill）
    │
    └─ 退出程序
```

### 会话并发控制

- **目的**: 防止同一个 (chat_id, agent_name) 的并发调用导致会话冲突
- **实现**:
  - `session_locks = {(chat_id, agent_name): threading.Lock}`
  - `_session_locks_creation_lock` 保护字典本身的并发访问
- **流程**:
  ```
  run_agent_cli()
      │
      ├─ 获取创建锁
      │  └─ 检查并创建会话锁
      │
      ├─ 获取会话锁
      │  └─ 执行 AI CLI
      │     └─ 保存会话
      │
      └─ 释放会话锁
  ```

## 启动参数

### agora 脚本参数

```bash
agora [-p 项目路径] [-r 轮次] [-s 分数]

参数:
  -p PATH    指定项目路径（默认：当前目录）
  -r ROUNDS  设置最大讨论轮次（默认：5，最大：10）
  -s SCORE   设置收敛分数阈值（默认：90）
  -h         显示帮助

示例:
  agora                      # 默认配置
  agora -r 10 -s 100         # 10轮讨论，100分收敛
  agora -p /path/to/project  # 指定项目路径
```
